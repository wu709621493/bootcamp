# tan 与 cot：贪与不贪的策略思考

本文尝试用三角函数 `tan` 与 `cot` 的互补关系，类比在博弈或投资中「贪与不贪」的决策逻辑，并讨论借款上限（borrow limit）、下注上限（bet limit）与回本困境（break-even dilemma）之间的互动。

## 1. tan 与 cot 的互补关系

- `tan(θ)` 随着角度接近 \(\frac{\pi}{2}\) 时趋向无穷大，象征追求极端收益时风险也急剧上升。
- `cot(θ)` 则在角度接近 0 时趋向无穷大，强调在极端保守情境下，机会成本会迅速扩大。
- 两者互为倒数：`tan(θ) = 1 / cot(θ)`，提醒我们贪婪与克制需要动态平衡，而非非黑即白。

## 2. 贪与不贪：风险报酬的平衡

1. **贪（高 `tan` 区域）**
   - 追逐高倍收益，常伴随高波动与高爆仓概率。
   - 决策上应明确止损位，并确认资金成本能否覆盖潜在损失。
2. **不贪（高 `cot` 区域）**
   - 偏向稳健，但若过度保守，则收益率被摊薄，可能错失成长机会。
   - 关键在于利用数据验证风险假设，避免无谓的机会成本。

## 3. Borrow Limit：借款上限的杠杆约束

- 借款上限设定了最大可动用的外部资金，等同于限制 `tan(θ)` 的最大可容忍值。
- 当借款使用率接近上限时，任何波动都会放大心理压力。
- 建议将借款率与自身资金比值控制在可承受范围，并预留风险缓冲区。

## 4. Bet Limit：下注上限的纪律防线

- 下注上限将单次投入限制在可控范围，类似在 `cot(θ)` 提供的稳定区间内运作。
- 它防止短时间内过度贪婪，也提醒在连赢时保持冷静。
- 可参考 Kelly 准则或分段下注法，根据胜率调整下注规模。

## 5. 回本困境（Break-even Dilemma）

- 当连续亏损后，很多人希望「回本」，往往驱使 `tan` 持续增大（加码追损）。
- 然而借款上限与下注上限会形成夹击：
  - 借款上限让可投入资金缩小。
  - 下注上限让单次翻盘难度加大。
- 若忽视这些约束，容易陷入「越补越亏」的恶性循环。

## 6. 策略建议：在 tan 与 cot 之间滑翔（op fly）

1. **动态调整风险角度 θ**
   - 依据市场波动与自身资金情况，调整 `θ` 落在 `tan` 与 `cot` 的平衡区间。
2. **建立双重限额体系**
   - 同时设置借款与下注上限，并根据盈亏自动收紧或放宽。
3. **强调回本不是唯一目标**
   - 把焦点放在长期期望报酬，而非短期回本。
4. **记录与复盘**
   - 用数据追踪每次调整 `θ` 的结果，确保策略在统计上可行。

## 7. 结语

`tan` 与 `cot` 的互补提醒我们：贪与不贪并非对立，而是需要在不同情境下做出的权衡。掌握借款上限与下注上限，理解回本困境的心理陷阱，才能在资金曲线中「滑翔（op fly）」，找到属于自己的稳健轨迹。
