# 搞笑招式：吞手（基于云手）

## 招式理念
- **原型启发**：取太极云手的圆柔路线，但将双手化作“吞云吐雾”的夸张动作，突出喜剧感。
- **核心卖点**：观者以为要接云手，结果看到使用者像吞云吞雾般“吞”自己的手，再把手从袖口另一端“吐”出，形成戏谑反差。

## 练习步骤
1. **起势换云**：站成太极起势，手随身转画圆，同步做深呼吸，好像在聚集“云雾”。
2. **吞入袖中**：当双手划到胸前时，突然将右手快速从袖口伸入衣袖，表情保持肃穆。
3. **云雾翻腾**：左手继续做云手轨迹，右臂在袖中悄悄移动到另一袖口，同时发出“呼——咕噜”的夸张音效。
4. **吐手成云**：右手从左侧袖口缓慢探出，手指张开，像放出一团云；配合身体轻转，完成一圈云手。
5. **循环反复**：两侧交替“吞”“吐”，还可加入假装卡手、找不到出口等即兴表演增加笑点。

## 实战效果（搞笑版）
- **扰乱节奏**：突如其来的“吞手”动作让对手犹豫，给出短暂的心理空档。
- **表演互动**：适合舞台、短视频或团队破冰，观众可以学着一起“吞云吐手”。

## 安全提醒
- 保持袖口宽松，避免卡手。
- 音效和表情尽量夸张但身体放松，避免拉伤肩肘。

## 搭配变化
- **烟雾道具**：袖中藏安全烟雾纸或彩带，吐手时带出更具视觉冲击。
- **节奏版**：配合鼓点或口技，每次吞吐动作对应强拍，节奏感更鲜明。
- **团体版**：多人围圈做云手，按顺序吞吐形成“云轮传递”的连续滑稽画面。
