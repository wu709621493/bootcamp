# 原则性问题往往是好问题

提出原则性问题，往往可以在复杂情境中明确出核心矛盾和判断标准。以下几个角度说明了这一点：

1. **澄清价值取向**：原则性问题迫使人们回到根本目标和优先级，从而避免在细节中迷失。
2. **暴露假设条件**：对原则的追问能帮助团队识别潜在的默认前提，减少“理所当然”的盲点。
3. **提升决策一致性**：当对原则达成共识后，后续的具体决策就能快速对齐，降低内耗。
4. **促进可复用经验**：被原则验证过的思考框架更易迁移，便于在类似问题上复用。
5. **引导建设性讨论**：围绕原则讨论，往往能把争论引向“为什么”，而不仅是“怎么做”。

简言之，抓住原则性问题，是在快速找到问题的“支点”，以最小的力气撬动最大的改进。
