# 中国书信与函件沟通的风格指南

本指南用于规划在中国语境下的正式与半正式书面沟通（纸质信函、电子邮件、公函、公告等）。重点涵盖语气、结构、书写习惯与视觉要素，帮助在跨机构、政务、企业与社会组织沟通中体现稳健、礼敬与执行力。

## 核心风格原则
- **礼敬与自抑**：以对方为中心，避免直白命令或过度自我凸显，用“请”“烦请”“敬请”等降低语气强度。
- **稳健与确定性**：表述政策、时间、责任和风险时使用明确边界与条件，避免模糊承诺。
- **层级分明**：先立场后细则，先背景后需求，先感谢后行动，段落之间递进。
- **简练与可执行**：主语清晰，时间、地点、责任人可落地，避免冗长修饰语。
- **合规与留痕**：在政务或商务场景注明文号、附件、抄送对象与联系人，便于追溯。

## 结构模板
1. **抬头与称呼**：
   - 对政府/机构：使用正式全称与敬语，如“××省××厅（局）：”“尊敬的××领导：”。
   - 对企业/伙伴：使用公司全称或职务敬称，如“××公司××总：”。
2. **开场致意**：简短问候与感谢，引用对方近期行动或合作背景建立联系。
3. **写信缘由**：一句话说明目的（咨询、报告、协调、反馈、邀请、确认等）。
4. **主体内容**：
   - **背景**：概述现状、依据或相关政策文件。
   - **事项/需求**：分点列出要做的事、责任分工、时间节点。
   - **约束与风险**：说明前提条件、合规要求或潜在影响。
5. **行动请求**：明确需要对方提供的行动、材料或回复期限。
6. **感谢与期待**：表达感谢与后续联络意愿。
7. **落款**：单位/姓名、职务、联系方式、日期；正式公函附文号、盖章位置。

## 用语与格式
- **敬语搭配**：
  - 开场常用：“您好！感谢贵单位/贵司在……的支持。”“承蒙关注，特此函告。”
  - 行动请求：“请贵方确认”“烦请安排”“敬请审阅并反馈”。
  - 结尾致意：“顺颂商祺”“敬祈垂注”“此致敬礼”。
- **时间表达**：使用“于×年×月×日前”“自××起生效”，避免“尽快”“近期”这类含糊词。
- **数字与单位**：遵循国家标准（GB/T 15835）书写阿拉伯数字与计量单位，金额用人民币大写避免歧义。
- **标点与段落**：中文全角标点，段首不缩进或两字符缩进保持一致，条列用“（一）（二）”“1. 2.”或“•”。
- **附件标注**：正文末尾列出附件名、份数与页数；邮件正文中提示“附件为……，请查收”。
- **抄送/密送**：在公函显著位置标注抄送单位；电邮区分“抄送”“密送”并写明理由。

## 场景化样板
### 公函（请示/商请）示例
```
××省××厅：

您好！为落实《××方案》相关要求，我单位拟于2024年9月启动××项目。现就以下事项请予协调：

（一）请协助确认××数据库的接入规范与安全测评时间表；
（二）烦请安排技术对接联系人，便于在9月15日前完成接口联调；
（三）如需额外合规审查，请告知材料清单和受理时限。

附件：××项目概要 1 份 4 页。

顺颂商祺！

××单位（盖章）
联系人：张三 138-0000-0000
2024年8月30日
```

### 企业合作邮件示例
```
尊敬的××总：

感谢贵司上周就××合作框架的讨论。现确认双方下一步安排：
1. 我方将在9月5日前提交技术方案与报价；
2. 请贵司在9月10日前反馈接口安全要求及测试窗口；
3. 建议9月12日组织线上评审会议，由双方技术、法务参与。

附件：技术方案初稿（v0.9）。如有补充需求，敬请告知。

此致
敬礼

李四
职务/部门 | 公司名称
电话 | 邮箱
```

## 视觉与版式建议
- **纸质公函**：使用统一页眉（单位名称、文号）、A4 纵向，正文宋体三号或小四，行间距固定 28 磅左右；章戳居中或右下。
- **电子邮件**：主题前添加标签（如“【请确认】”“【协调】”），正文采用 12–14 px 易读字体，保持 60–70 字符每行；重要日期加粗。
- **品牌一致性**：对外沟通遵循单位 VI（色彩、LOGO、签名档），多语言场景提供中英双语封面或摘要。

## 审核与风险控制清单
- 事项、时间与责任是否明确且可追溯；
- 引用的政策、合同或数据是否标注来源与版本；
- 附件齐全且文件名规范；
- 抄送对象符合信息最小分发原则；
- 涉及敏感信息是否脱敏或获取授权；
- 日期、数字与单位格式是否符合国家标准。

通过以上原则与模板，可以快速生成符合中国语境的专业、可靠、可执行的书面沟通材料。
